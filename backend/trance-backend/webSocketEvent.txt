

requested Events:

send-request : send a request to a user. (done ...)

accept-request: when a user accept a friend request

refuse-request: when a user refuse a friend request

listened Events:

statusChange : when a user status change to something else

error : when an error occurs with web socket i send a string displying the error

request-sent: sent to the sender socket to inform that the request has been sent sucffeuly

friend-notif: sent to the user if online to notify him of a friend request

request-accepted: sent to the sender to notify him of the friend accepting the request

request-refused: sent to the sender to notify him of the friend refusing the request
